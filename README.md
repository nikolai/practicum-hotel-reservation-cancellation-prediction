# Модель предсказания вероятности отказа от бронирования номера в отеле

## Описание

Ключевые слова: бинарная классификация, бустинг, векторизация текста

## Требования

- **Python**: 3.11 или выше
- **ОС**: Linux, macOS или Windows (с WSL)

## Установка и запуск

### Конфигурация подключения к БД

Создайте файл `.env` на основе шаблона `.env.example`:
Затем отредактируйте `.env` и укажите ваши данные для подключения к PostgreSQL.

### Для Fish shell:

```fish
fish start.fish
```

### Для Bash/Zsh:

```bash
./start.sh
```
